# DMM Frontend (React)

Isolated SPA for Digital Marketing Manager.

## Env
Create .env as:
```
REACT_APP_BACKEND_URL=https://dmm.yourdomain.com
```

## Pages
- Strategy (stub)
- Content: Reels, UGC, Brand, Influencer
- Campaigns
- Approvals

## Dev
```
yarn
yarn dev
```

All API calls use REACT_APP_BACKEND_URL + /api.
